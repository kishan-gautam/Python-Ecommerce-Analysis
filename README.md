# Python-Ecommerce-Analysis
# 🛒 Ecommerce Analysis — Python & Pandas

> Exploratory data analysis on 10,000 ecommerce transactions using Python and Pandas.  
> Practiced real-world data analysis questions as part of my Python learning journey.

---

## 📁 Dataset

| Feature | Details |
|---|---|
| Rows | 10,000 |
| Columns | 14 |
| Source | Ecommerce Purchases CSV |

**Columns include:** Address, AM/PM, Browser Info, Company, Credit Card, CC Provider, Email, Job, IP Address, Language, Purchase Price

---

## ❓ Questions Answered

1. Display the dataset
2. Display top 10 and last 10 rows
3. Check datatype of each column
4. Check for null values
5. How many rows and columns are there?
6. What is the highest and lowest purchase price?
7. What is the average purchase price?
8. How many people have French (`fr`) as their language?
9. How many people have Engineer as their job title?
10. Find the email of the person with IP Address `132.207.160.22`
11. How many people used Mastercard and purchased over $50?
12. How many people purchased during AM vs PM?
